
# 50 - [Pyevmasm](./Pyevmasm.md)

[Pyevmasm](https://github.com/crytic/pyevmasm) is an assembler and disassembler library for the Ethereum Virtual Machine (EVM). 

It includes a command line utility and a Python API.
